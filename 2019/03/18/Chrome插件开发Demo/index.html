Chrome插件开发Demo

发表于 2019-03-18 | 更新于 2019-03-20 | 分类于 Chrome | 阅读次数：


对于一个前端developer来说，chrome插件的开发非常简单，几个简单的文件就能轻松搞定。

插件的目录结构

创建一个目录(demo)，并在目录中创建配置文件manifest.json

创建弹出页面popup.html

创建执行脚本popup.js

qrcode.min.js文件下载自davidshimjs.

验证插件是否可用

打开chrome扩展程序chrome://extensions/，选择开发者模式，选择已解压的扩展程序，选择demo目录，然后就看到插件二维码生成器已经成功加载了

打包

打开chrome扩展程序chrome://extensions/，选择开发者模式，选择打包扩展程序，如果打包失败，请根据错误提示信息修正，如果打包成功，会在与根目录同级的目录中生成CRX插件文件和PEM秘钥文件。

安装扩展程序

打开chrome扩展程序chrome://extensions/，拖拽CRX文件进入到当前页面即可完成安装。
